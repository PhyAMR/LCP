# LCP - Laboratory of Computational Physics

This directory contains all materials related to the Laboratory of Computational Physics course.

## Folder Structure

- **`class_notes/`**: Personal notes from lectures.
- **`code/`**: Jupyter notebooks, Python scripts, and other code for labs and assignments.
- **`final_project/`**: Final project materials.
- **`professor_notes/`**: Lecture slides and materials from the professor.
